# college-library-data-usage
“Analyzing college library data: student visits, book borrowing, fines, and study patterns.”


## Project Overview
Analyzing college library data including student visits, book borrowing, fines, and study patterns.

This project studies how students use the college library to understand borrowing habits, study behavior, visit frequency, and fine patterns using real dataset records.

---

## What This Project Does
- Cleans and organizes library usage data  
- Analyzes student borrowing behavior  
- Compares usage across departments and years  
- Studies relationship between study hours and visits  
- Identifies overdue books and fine patterns  
- Classifies students based on library usage level  

---

## Tools & Technologies Used
- Python — data processing  
- Pandas — data cleaning & analysis  
- Jupyter Notebook — interactive coding environment  

---

## Dataset Includes
- Student department & year  
- Membership type  
- Book genres  
- Books borrowed  
- Study hours & visit frequency  
- Borrow status & fines  

---

## Outcome
- Converts raw data into meaningful insights  
- Shows library usage trends  
- Identifies overdue risks  
- Improves practical data analysis skills  

---

## Conclusion
This project demonstrates how structured data analysis supports better academic decision-making and shows real-world use of Python in educational data analysis.
